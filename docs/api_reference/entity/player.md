## `Player` vs `PlayerRef`

Historiquement, `PlayerRef` était utilisé pour faire référence à une entité joueur. Cette approche est maintenant **dépréciée (`@Deprecated`)**.

La méthode moderne consiste à utiliser directement le **Composant `Player`**, qui contient toutes les données et la logique nécessaires. La nouvelle page de documentation est basée sur cette approche moderne.
