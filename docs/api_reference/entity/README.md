# Entity API Reference

This section provides detailed API documentation for managing entities within the Hytale server. This includes core entity concepts, player-specific data, and how to interact with them programmatically.

## Classes

-   [`PlayerRef`](./PlayerRef.md): A reference to a connected player, providing access to player-specific data and network interaction.
-   [`Player (Entity Component)`](./PlayerEntityComponent.md): The core entity component representing a player's in-game state, inventory, and interactions.
