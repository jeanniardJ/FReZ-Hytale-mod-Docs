# Référence API des Composants (ECS)

Cette section fournit la documentation détaillée pour les classes fondamentales du système d'entités basé sur les composants (ECS - Entity Component System) de Hytale. Ce système est au cœur de la manière dont les entités et leurs comportements sont définis et gérés dans le jeu.

## Classes

-   [`ComponentType`](./ComponentType.md) : Un identifiant unique et un conteneur de métadonnées pour chaque type de composant dans le système ECS.
-   [`Store`](./Store.md) : Le cœur du système ECS, gérant toutes les entités et leurs composants de manière performante et thread-safe.
