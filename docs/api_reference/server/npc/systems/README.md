# Référence API des Systèmes PNJ

Cette section fournit la documentation détaillée pour les classes qui implémentent divers systèmes d'entités liés aux PNJ (Personnages Non Joueurs). Ces systèmes définissent la logique et le comportement des PNJ dans le monde du jeu.

## Classes

-   [`NPCDeathSystems`](./NPCDeathSystems.md) : Gère la logique liée à la mort des PNJ, y compris les interactions avec d'autres entités et le suivi des événements.
-   [`BlackboardSystems`](./BlackboardSystems.md) : Contient les systèmes d'entités PNJ qui interagissent avec le tableau noir de l'IA.
-   [`NPCSystems`](./NPCSystems.md) : Un ensemble de systèmes ECS pour la gestion générale des PNJ.
