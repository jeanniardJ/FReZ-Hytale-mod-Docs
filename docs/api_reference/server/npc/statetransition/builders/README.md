# Référence API des Constructeurs de Transitions d'État PNJ

Cette section fournit la documentation détaillée pour les classes utilisées pour construire et définir les transitions d'état des PNJ (Personnages Non Joueurs). Ces constructeurs sont essentiels pour sculpter l'intelligence artificielle et le comportement des PNJ.

## Classes

-   [`BuilderStateTransition`](./BuilderStateTransition.md) : Un constructeur pour définir comment un PNJ passe d'un état de comportement à un autre.
-   [`BuilderStateTransitionEdges`](./BuilderStateTransitionEdges.md) : Un constructeur pour définir les arêtes des transitions d'état.
-   [`BuilderStateTransitionController`](./BuilderStateTransitionController.md) : Un constructeur pour gérer une collection de transitions d'état.
