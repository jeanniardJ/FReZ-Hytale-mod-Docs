# Référence API des Rôles de PNJ

Cette section fournit la documentation détaillée pour les classes et concepts liés aux "rôles" des PNJ (Personnages Non Joueurs). Un rôle définit le comportement, l'IA, les interactions et les caractéristiques d'un PNJ spécifique.

## Classes

-   [`Role`](./Role.md) : La classe centrale qui encapsule le comportement et les propriétés d'un PNJ.
-   [`RoleDebugFlags`](./RoleDebugFlags.md) : Définit les indicateurs de débogage pour les rôles de PNJ.
