# Référence API des Validateurs du Constructeur d'Assets PNJ

Cette section fournit la documentation détaillée pour les classes liées aux validateurs utilisés dans la construction des assets des PNJ (Personnages Non Joueurs). Ces validateurs garantissent la cohérence et la validité des données d'assets.

## Classes

-   [`SubTypeTypeAdapterFactory`](./SubTypeTypeAdapterFactory.md) : Une fabrique d'adaptateurs de type pour Gson, gérant la sérialisation/désérialisation polymorphique d'objets JSON.
