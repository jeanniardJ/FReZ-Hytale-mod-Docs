# Documentation des Événements

Le système d'événements est un pilier central du développement de mods sur Hytale. Il vous permet de créer des fonctionnalités qui réagissent dynamiquement à ce qui se passe dans le jeu.

Cette section contient toutes les ressources pour maîtriser les événements.

## Guides et Références

*   **[Concepts Clés : Le Système d'Événements](../general/event_system.md)**  
    *Apprenez les bases : ce qu'est un événement, comment l'écouter, gérer les priorités et annuler des actions. **À lire en premier !***

*   **[Liste des Événements Serveur](server_events_list.md)**  
    *Une liste de référence rapide des événements les plus courants (connexion de joueur, interaction avec les blocs, etc.) pour savoir ce que vous pouvez écouter.*
